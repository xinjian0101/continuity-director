# v0.4.0 二十轮运行时迭代

本文件记录从 v0.3.0 到 v0.4.0 的连续二十轮有效迭代。

1. 版本升级至 v0.4.0，Manifest Schema 升级至 1.3。
2. 增加参考帧生命周期注册表。
3. 增加多角色入场、在场、退场状态验证。
4. 增加精确到帧的时间线和时间码。
5. 增加镜头依赖 DAG 编译与循环检测。
6. 增加确定性并行执行波次。
7. 增加固定、递增和稳定变体三种重试种子策略。
8. 增加便携生成任务队列。
9. 增加异步任务结果回填与汇总。
10. 增加超时、限流、显存不足、审核和参数错误分类。
11. 增加声明式模型配置注册表。
12. 增加序列与模型能力匹配。
13. 增加安全模型提示词模板。
14. 增加按镜头、角色、场景和权重选择参考帧。
15. 增加生成工作量与配置成本估算。
16. 增加完整确定性执行计划编译。
17. 增加关键路径、并行率和参考覆盖诊断。
18. 增加 12 个 ComfyUI 运行时节点。
19. 自动化测试扩展至 56 项。
20. 完成中英文文档、Schema、示例和发布校验。

所有运行时功能只编译本地 JSON 数据，不直接调用网络服务，不读取或保存 API Key。
